Descripción del Proyecto

Este versión de equipo, diseñado manejar 2 embolsadoras de manera simultánea, podrá cargar distintas funciones y datos de acuerdo a la necesidad de producción.

Trabajo en equipo

Para alcanzar los niveles de precisión y control requeridos en este proyecto, fue esencial el uso de un microcontrolador. Colaboré con un compañero encargado de programar este microcontrolador, que gestionaba la balanza utilizada para llenar bolsas con precisión. Mi función principal fue coordinarme con él para asegurar una integración fluida entre el sistema Android y el microcontrolador. Esto implicaba una comunicación continua para transmitir información sobre los cortes y el estado del proceso, así como para enviar los valores de trabajo y las configuraciones necesarias para el operario. Mi objetivo era garantizar que los datos se presentaran correctamente al operador y que el microcontrolador recibiera la información precisa para controlar eficazmente los procesos de corte.
