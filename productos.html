Descripción del Proyecto

Este programa esta diseñado para trabajar con fórmulas de producción. Apto para todos aquellos procesos que requieran el uso de distintos ingredientes y varios pasos, dentro del proceso de producción. El indicador permitirá cargar todos los productos involucrados y luego confeccionar las distintas recetas, cargar turnos de trabajo, tolerancia de llenado, reportes varios, etc.

Tecnologías Utilizadas

  • Java
  • Android Studio
  • SQLite
